# Dark Patterns in Augmented Reality – Educational AR Application

## Project Overview

This project was developed as part of a master's thesis and explores how users can be educated about Dark Patterns in Augmented Reality through immersive and interactive learning experiences.

The application was developed for the Meta Quest 3 and combines educational content, interactive AR scenarios, and contextual explanations to help users recognize and better understand manipulative design strategies in AR environments.

The project includes four interactive scenarios demonstrating different Dark Patterns in Augmented Reality, embedded within a structured learning application that introduces the concept of Dark Patterns and explains the manipulation techniques after each scenario.

Additionally, as part of the project, a solution was developed for persistent spatial content placement using spatial anchors.

The application was developed through a user study with participants.

---

## Concept

The application was designed as an interactive educational experience that combines:
- theoretical explanations
- immersive AR interaction
- scenario-based learning
- reflective post-scenario explanations

Users first receive an introduction to Dark Patterns and their relevance in AR environments. Afterwards, they experience multiple interactive scenarios in which manipulative interface strategies are applied. After each scenario, the application explains the implemented manipulation techniques and their intended psychological effects.

---

## Interactive Scenarios

The application includes four interactive Dark Pattern scenarios:

- Attention Grabber
- Interface Obstruction
- Strategic Content Placement
- Associative Manipulation

Each scenario was specifically designed for immersive AR interaction and demonstrates how manipulative design techniques can influence user behavior in Augmented Reality environments.

## Challenges

One of the major challenges was designing Dark Pattern interactions that were both realistic and educational without frustrating participants excessively.

Additional challenges included:
- persistent placement of virtual content in real environments
- balancing immersion and usability
- creating intuitive AR interactions
- designing educational explanations for manipulative behavior
- ensuring stable spatial anchor behavior on the Meta Quest 3

---

## Educational Goal

The goal of this project was to investigate how immersive educational experiences in Augmented Reality can improve users' awareness and understanding of Dark Patterns in AR environments.

The project explored whether interactive and scenario-based learning approaches can help users recognize manipulative interface strategies more effectively than traditional explanatory methods.
